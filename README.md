# Nigeria-Inflation-Analysis-2003-2026
A Python-based study of Nigerian inflation trends, focusing on economic shocks and purchasing power.  

🚀 Key Highlights
Peak Inflation: Reached an average of 33.18% in 2024.  

Economic Shock: Largest monthly spike (+7.19) occurred in December 2024.  

Current Trend: Inflation is cooling, averaging 15.18% in Q1 2026.  

Income Impact: A 175,000 NGN salary currently has a real value of ~158,227 NGN.  

🛠️ Tools Used
Analysis: Python (Pandas).  

AI Insights: Qwen 2.5:7b via Ollama.  

Reporting: Automated PDF generation via FPDF.  

📂 Project Structure
inflation_analysis.ipynb: Data processing and calculations.  

Inflation_data.csv: Historical dataset.  

Inflation_Analysis_Report_v1.pdf: Final executive summary.
